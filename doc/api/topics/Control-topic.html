Control Topic

These classes ensures the processing required for the execution of a use-case and its business logic, and coordinates, sequences controls other objects involved in the use-case.

Classes

CarparkInfoManager Control
This class is responsible for providing CarparkInfo data, loaded from the CSV obtained from the Singapore government. This data is static information about the carparks like their addresses, LatLng info and unique carpark codes.
CarparkListManager Control
Handles list construction for CarparkListView
DatabaseManager Control
The class to interface with the device's local SQL database.
LocationManager Control
The class responsible for all location queries.
MapViewWithSearch Control
The control class that handles the location searching. Comprises of the mapview as well as the autocompleting search bar.
MarkerGenerator Control
A marker icon generator class. As the GoogleMaps marker class only accepts a BitMap Descriptor, we use this class to turn our Icons classes to BitMapDescriptors.
PullDateManager Control
Handles the logic to pull dates that are missing from the database and delete dates that are outside the window.
ScreenManager Control
This control class is responsible for displaying the appropriate screens by calling the right route functions.
TabsManager Control
This class controls the view between the MapViewWithSearch and the CarparkListView by providing a BottomNavigationBar.
